jquery.todoList - Demo

jQuery TodoList plugin allows you to insert tasks into page, that can be checked off as work on them completes. Click any area with text to edit the title of that element.

The todo list's configuration can be retrieved via the API so that it can be stored and used on subsquent visits by the user to re-create the list.
